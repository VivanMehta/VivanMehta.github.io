GITHUB PAGES FIXED VERSION

1. Open your GitHub repository:
   https://github.com/VivanMehta/vivanmehta

2. Delete the current website files, or upload these files and choose to replace files with the same names.

3. Upload the CONTENTS of this folder directly to the repository root:
   - index.html
   - 404.html
   - favicon.ico
   - assets folder
   - .nojekyll
   - README.md

4. Commit the changes.

5. Wait about 1–3 minutes, then hard-refresh:
   Windows: Ctrl + Shift + R

This version changes /assets/... references to /vivanmehta/assets/... so animations,
fonts, images, and JavaScript load correctly from a GitHub project site.
